# Blood Pressure Chart ŌĆö Design Spec

**Date:** 2026-06-12

## Overview

Add a min-max range chart to the blood pressure section (`/pressione`) that visualises systolic/diastolic pressure as floating bars and pulse as a dashed line, with a preset date-range selector.

## Layout

The page order becomes:

1. Chart (`BloodPressureChart`) ŌĆö new
2. Add measurement form ŌĆö unchanged
3. Measurements list ŌĆö unchanged

Max width stays `max-w-lg` matching the existing page layout.

## New Component: `BloodPressureChart`

**Location:** `frontend/src/components/BloodPressureChart.tsx`

**Props:**
```ts
interface BloodPressureChartProps {
  measurements: BloodPressureRecord[]
}
```

The component receives the already-loaded measurements array from `Pressione.tsx` (no additional API calls). All date filtering is done client-side.

### Date Preset Selector

Four preset buttons rendered above the chart:

| Label | Window        |
|-------|--------------|
| 7gg   | Last 7 days  |
| 30gg  | Last 30 days |
| 3m    | Last 3 months|
| Tutto | No filter    |

- Default preset: **30gg**
- Active preset: `Button` variant `default`
- Inactive presets: `Button` variant `outline`
- Preset state is local to `BloodPressureChart` ŌĆö not exposed to the parent

If the filtered dataset is empty, show a text placeholder instead of the chart (same `text-muted-foreground` style used in the list).

### Chart

Uses shadcn/ui `chart` component wrapping Recharts `ComposedChart`.

**Data transformation** (before passing to Recharts):
```ts
{
  date: string        // formatted label: always "dd/MM HH:mm" so same-day measurements are distinguishable
  diastolic: number   // base of the floating bar (transparent)
  range: number       // systolic - diastolic (visible bar height)
  pulse: number | undefined
}
```

**Recharts composition:**
- `Bar` (stack id `"bp"`, `fill="transparent"`, `dataKey="diastolic"`) ŌĆö invisible base
- `Bar` (stack id `"bp"`, `dataKey="range"`, `fill="var(--chart-1)"`) ŌĆö visible range bar
- `Line` (`dataKey="pulse"`, `strokeDasharray="4 2"`, `yAxisId="right"`, `stroke="var(--chart-2)"`) ŌĆö pulse dashed line
- `YAxis` left ŌĆö label "mmHg", domain auto-fitted to data
- `YAxis` right (`yAxisId="right"`) ŌĆö label "bpm"
- `XAxis` ŌĆö `dataKey="date"`
- `Tooltip` ŌĆö shows sistolica, diastolica, battiti

**Colors:** shadcn/ui CSS variables (`--chart-1`, `--chart-2`) so they adapt to light/dark theme automatically.

## Dependencies to Add

- `recharts` (npm package, Recharts v2)
- shadcn/ui `chart` component: `npx shadcn@latest add chart`

## Out of Scope

- Custom date range picker (free input) ŌĆö preset buttons only
- Reference zones (e.g. green band for normal pressure) ŌĆö not requested
- Pulse on a separate chart ŌĆö same chart, right Y axis
- Extracting form or list into separate components
